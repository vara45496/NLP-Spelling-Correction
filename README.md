# NLP Spelling Error Detection and Correction

##  Project Description
This project focuses on detecting and correcting spelling errors using NLP techniques using the Birkbeck dataset.

##  Methods Used
- Edit Distance
- Noisy Channel Model
- SymSpell Algorithm
- Phonetic Algorithms (Soundex, Metaphone)

##  Evaluation Metrics
- Accuracy
- Top-1 Accuracy
- Top-5 Accuracy
- Mean Reciprocal Rank (MRR)

##  Dataset
- Birkbeck Spelling Error Dataset

python main.py
